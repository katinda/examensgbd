# Dossier models/

Ce dossier contient les classes qui représentent les tables de la base de données.

Chaque fichier = une table.
Exemple : Site.php représente la table Sites.

Ces classes contiennent uniquement des propriétés (les colonnes de la table)
et des getters/setters pour y accéder.
Elles ne font jamais de SQL et n'ont aucune logique métier.
